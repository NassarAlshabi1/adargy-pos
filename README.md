## نظرة عامة

تطبيق **الهيثم للتجارة** هو نظام متكامل مصمم لمساعدتك على إدارة متجرك بما في ذلك المبيعات، المنتجات، العملاء، الموردين، الديون، والمصروفات. تم تطويره باستخدام إطار العمل Flutter، مما يضمن تجربة مستخدم سلسة وأداءً عالياً على مختلف المنصات. يعمل التطبيق بشكل كامل بدون الحاجة للاتصال بالإنترنت، حيث يتم تخزين جميع البيانات محلياً على جهازك لضمان الخصوصية والأمان.

---

## المميزات الرئيسية

- **إدارة المبيعات**: الدفع النقدي، التقسيط، والبيع الآجل
- **إدارة المنتجات والمخزون**: تتبع المنتجات والكميات المتاحة
- **إدارة العملاء**: إضافة وتعديل بيانات العملاء ومتابعة الحسابات والديون
- **إدارة الموردين**: التعامل مع الموردين والمشتريات
- **تسجيل المدفوعات**: تسجيل الدفعات والمرتجعات
- **تقارير شاملة**: تقارير PDF للمبيعات والمخزون والأرباح
- **إدارة الديون**: متابعة الديون والأقساط والتحليلات
- **التصنيفات والأكواد**: تنظيم المنتجات وإدارة عروض الخصم
- **رسوم بيانية وتحليلات**: عرض ملخصات مالية تفصيلية
- **دعم اللغة العربية**: واجهة كاملة من اليمين لليسار (RTL)
- **دعم الأوفلاين**: يعمل بدون إنترنت مع تخزين محلي آمن

## التقنيات المستخدمة

- **إطار العمل**: [Flutter](https://flutter.dev/)
- **لغة البرمجة**: [Dart](https://dart.dev/)
- **إدارة الحالة**: [Provider](https://pub.dev/packages/provider)
- **قاعدة البيانات**: [SQFlite](https://pub.dev/packages/sqflite)
- **إنشاء PDF والطباعة**: [pdf](https://pub.dev/packages/pdf) & [printing](https://pub.dev/packages/printing)
- **الرسوم البيانية**: [fl_chart](https://pub.dev/packages/fl_chart)
- **تشفير البيانات**: [crypto](https://pub.dev/packages/crypto)
- **توليد الباركود**: [barcode_widget](https://pub.dev/packages/barcode_widget)

## بدء الاستخدام

لتشغيل هذا المشروع على جهازك، اتبع الخطوات التالية:

1. **تثبيت Flutter**:
   تأكد من أن لديك [Flutter SDK](https://flutter.dev/docs/get-started/install) مثبت على جهازك.

2. **نسخ المشروع**:
   ```bash
   git clone https://github.com/NassarAlshabi1/adargy-pos.git
   cd adargy-pos
   ```

3. **تثبيت الاعتماديات**:
   قم بتشغيل الأمر التالي في الطرفية (Terminal) لتثبيت كل الحزم المطلوبة:
   ```bash
   flutter pub get
   ```

4. **تشغيل التطبيق**:
   اختر الجهاز الذي تريد تشغيل التطبيق عليه (محاكي أو جهاز حقيقي) ثم قم بتشغيل الأمر:
   ```bash
   flutter run
   ```

## السيناريوهات المدعومة

### البيع الآجل مع السداد الجزئي
يدعم النظام السيناريوهات المعقدة مثل:
- بيع آجل بقيمة 50,000
- تسديد 20,000
- ثم بيع آجل آخر بقيمة 70,000

النظام يحسب الرصيد المتبقي تلقائياً:
```
إجمالي الديون: 120,000
إجمالي المدفوعات: 20,000
═══════════════
الرصيد المتبقي: 100,000
```
